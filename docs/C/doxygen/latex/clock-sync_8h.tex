Statistics for a given socket. The R\+TI initiates a clock synchronization action by sending its current physical time T1 to a federate. The federate records the local time T2 that it receives T1. It sends a reply at local time T3, which the R\+TI receives at its time T4. The R\+TI sends back T4. The round trip delay on the socket is therefore estimated as\+: (T4 -\/ T1) -\/ (T3 -\/ T2).
