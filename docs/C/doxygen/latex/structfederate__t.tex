\doxysubsection{Detailed Description}
Information about a federate known to the R\+TI, including its runtime state, mode of execution, and connectivity with other federates. The list of upstream and downstream federates does not include those that are connected via a \char`\"{}physical\char`\"{} connection (one denoted with $\sim$$>$) because those connections do not impose any scheduling constraints.
