\textquotesingle{} and \textquotesingle{}\textbackslash{}0\textquotesingle{}, where the latter registers a trigger to invoked when any key is pressed. If a specific key is registered and any key (\textquotesingle{}\textbackslash{}0\textquotesingle{}) is also registered, the any key trigger will be scheduled after the specific key is scheduled.
